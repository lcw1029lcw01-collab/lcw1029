# 02. VISION — 우리가 가는 곳

## 한 문장 비전

> **"주제 하나만 입력하면, 넷플릭스 수준의 AI 다큐멘터리가 자동으로 나오는 운영체제."**

사람은 주제 하나만 입력한다. 나머지는 전부 ADOS가 한다.

---

## 최종 목표 (사용자 경험)

터미널에서:

```bash
ados create \
  --topic "100만 년 후 인간은 어떤 모습일까?" \
  --language en \
  --duration 25
```

실행하면:

```
✔ 리서치 완료
✔ 대본 생성
✔ Scene 생성 (약 200개)
✔ 감독 연출 생성
✔ Midjourney Prompt 생성
✔ Kling Prompt 생성
✔ Typecast 생성
✔ 자막 생성
✔ Premiere XML 생성
✔ 썸네일 생성
✔ SEO 생성

출력 완료
```

---

## 왜 우리는 다른가

지금 AI 유튜브 하는 사람들:

```
GPT → 대본 → Midjourney → 영상 → 업로드   (끝)
```

우리:

```
AI Documentary OS → 감독 → 영화 → 자동 제작 → 수십 개 채널
```

**그들은 영상을 자동화한다. 우리는 회사를 자동화한다.**

---

## 채널 포트폴리오

같은 엔진으로 여러 채널을 운영한다. 채널은 서로 경쟁하지 않는 **하나의 AI 미디어 회사 포트폴리오**다.

### 1차 운영 채널 (선정 완료)

| 순위 | 채널 | 주제 | 비고 |
|------|------|------|------|
| 1 | **Beyond Humanity** | 미래 문명·인류·과학·철학 | 최우선. 첫 에피소드 제작 대상 |
| 2 | **Humanity Explained** | 인간 심리·행동·진화 | |
| 4 | **Civilization Stories** | 국가·문화·문명의 성공과 실패 | |

> 각 채널은 영어 채널 + 한국어 채널 2개로 운영한다.
> 영어 대본 작성 → 영어 Typecast → 한국어 번역(직역 금지, 자연스럽게) → 한국 Typecast → 같은 영상 재활용.

### Beyond Humanity 채널 정의

- 한 줄: **"인류의 미래, 우주의 끝, AI 문명, 그리고 인간 이후의 세상을 영화처럼 보여주는 채널."**
- 슬로건: "오늘의 인간을 넘어, 내일의 문명을 탐험하다." (EN: Exploring Humanity Beyond Tomorrow)
- 형식: 넷플릭스 다큐 + BBC + 미래 SF + 과학 + 철학. 모든 영상은 "실제로 일어날 수도 있는 이야기"
- 길이: 롱폼 20~35분 (기준 25분)
- 타깃: 18~44세, 미국·유럽·인도 중심 글로벌, 과학·AI·우주 관심층

### 확장 채널 (엔진 재사용)

History Reimagined, The Last Day on Earth, Future Earth, Space Civilization,
Future Technology, Future Economy, What If...? Universe → **Story Engine만 교체하면 된다.**
