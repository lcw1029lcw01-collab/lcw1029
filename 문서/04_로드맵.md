## Sprint 2 — Story Engine

- 25분 대본 구조 자동 생성 (Hook → Conflict → Discovery → Twist → Ending → Question)
- Scene Script 방식 (글이 아니라 영화로)
- World Bible 연동 (모든 영상이 같은 세계관)

## Sprint 3 — Director Engine

- 감정 → 카메라·렌즈·조명·색감·음악 자동 결정
- Shot 다양화 (같은 카메라 반복 금지)
- Emotion Engine 수치 모델

## Sprint 4 — Prompt Engine

- Midjourney Prompt 자동 조립 (Subject + Environment + Camera + Lens + Lighting + Mood + Film)
- Kling Motion Prompt 생성
- Character Bible 기반 인물 일관성

## Sprint 5 — Automation (Claude Code)

- 주제 입력 → JSON 생성 → Scene 생성 파이프라인
- Midjourney / Kling API 자동 연결
- Typecast 음성 길이 계산 → Scene 길이 자동 계산 → Premiere 배치

## Sprint 6 — Beyond Humanity Episode 1

- 첫 번째 영화급 다큐멘터리 완성
- 후보: "What Will Humans Look Like in 1 Million Years? (100만 년 후 인간은 어떤 모습일까?)"
