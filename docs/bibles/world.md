# World Bible (v0.1 초안)

> 모든 영상이 공유하는 세계관. 시청자는 영상이 아니라 **하나의 연속된 미래 역사**를 따라간다.
> 이것이 다른 AI 채널과의 결정적 차별점이다.

## 세계관 구성 요소

세계관은 다음 축으로 정의한다:

```
연도 → 기술 수준 → AI 수준 → 정치 → 경제 → 도시/건축 → 인간의 모습 → 패션 → 교통 → 음식 → 문화
```

## Beyond Humanity 미래 연대기 (마스터 타임라인)

| 연도 | 사건 |
|------|------|
| 2050 | AI가 대부분의 사무직을 대체한다 |
| 2065~2068 | AI 정부가 등장한다 |
| 2070~2080 | 달 식민지 |
| 2095 | 인간과 AI 시민권 논쟁이 시작된다 |
| 2100 | 화성 도시 |
| 2140 | 최초의 화성 독립국이 선언된다 |
| 2400 | 지구는 관광지가 된다 |
| 2500 | 태양계 전역 진출 |
| 3000 | 지구는 더 이상 인류의 중심이 아니다 |
| 1,000,000 | 인간 진화 (신인류) |

모든 에피소드는 이 타임라인 위의 한 지점을 다룬다. 영상 1이 2050년을 다루고, 영상 63이 2140년을 다뤄도 **같은 우주**다.

## Character Bible — Future Human Type-A

100편 동안 같은 캐릭터를 유지하기 위한 기준:

```
Future Human Type-A

Height:        190cm
Skin:          Transparent bio skin
Eyes:          No visible pupils
Hair:          None
Suit:          Nano fabric
Emotion:       Calm
Walking Style: Slow confident
```

> 새 캐릭터가 필요하면 여기에 Type-B, Type-C로 추가한다. 즉흥 생성 금지.

## Color Bible — 장소·상황별 색감

| 대상 | 색감 |
|------|------|
| Earth (지구) | Blue |
| Mars (화성) | Orange |
| Moon (달) | Gray |
| Future (미래 일반) | Cyan |
| Danger (위험) | Red |
| Memory (기억/회상) | Warm |

기본 톤: 항상 영화 느낌 — 푸른색, 검정, 주황, 네온. 채널 색감: 청록 + 주황 계열.

## Visual Motif (매 영상 반복)

- AI가 만든 **지구 홀로그램**
- 미래 연도를 표시하는 **HUD 인터페이스**
- "Simulation Start..." 오프닝 애니메이션
- 같은 내레이터 톤, 같은 배경 음악 테마

시청자가 몇 초만 봐도 "아, 이 채널 영상이다"라고 인식하게 만든다.
