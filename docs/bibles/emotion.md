# Emotion Bible (v0.1 초안)

> ADOS의 영혼. 감정이 모든 연출 결정의 시작점이다.

## 원칙

- **감정이 먼저다.** 카메라는 감정을 표현하기 위한 도구다.
- Scene마다 감정을 수치(0~100)로 표현한다.
- 가장 높은 감정이 연출을 지배하고, 보조 감정이 디테일을 만든다.

## 감정 수치 모델

```
Scene: 인류 마지막 날

외로움  80
희망    10
공포    60
신비    70
```

→ 지배 감정: 외로움 → Wide / Blue / Fog / Slow
→ 보조 감정: 신비 → Fog 강화 / 공포 → 어두운 그림자 디테일

## 감정 → 연출 규칙표

| 감정 | 연출 체인 |
|------|-----------|
| 공포 | 빨간색 → Close Up → Heartbeat(BGM) → Camera Shake |
| 희망 | Sunrise → Wide → Slow Piano |
| 외로움 | Wide Shot → 24mm → Blue → Fog/Rain → Slow Camera |
| 신비 | Fog → 낮은 채도 → 느린 카메라 |

> 이 표는 Sprint 3 (Director Engine)에서 전체 감정 스펙트럼으로 확장한다.
> 확장 대상: 경외감, 슬픔, 긴장, 호기심, 충격, 평온 등.

## 주제 감정 분석 (파이프라인 진입점)

주제가 입력되면 GPT가 먼저 감정을 분석한다:

```
주제: 100만년 후 인간

감정 70 / 신비 80 / 공포 20 / 희망 40
```

이 분석 결과가 Director Engine의 입력값이 된다:

```
신비 80 (지배) → Wide, Fog, Blue, 24mm, Slow
```
