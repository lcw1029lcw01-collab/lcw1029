# Director Bible (v0.1 초안)

> ⭐ ADOS의 두뇌. 감독이 하는 일을 AI가 한다. Sprint 3에서 정식 버전으로 확장한다.

## 감독의 정의

감독은 이미지를 만드는 사람이 아니다. **결정을 내리는 사람**이다.

```
이번 장면 → 비 와. → 카메라는 멀리 있어. → 색감은 차갑게. → 음악은 조용하게. → 5초 후 클로즈업.
```

결정 순서 (항상 이 순서):

```
감정 → 카메라 → 렌즈 → 색감 → 조명 → 음악 → 속도
```

**Prompt는 이 결정의 결과물이다. Director가 먼저다.**

## 감정 → 연출 매핑 규칙

| 감정 | 카메라 | 색/조명 | 연출 요소 |
|------|--------|---------|----------|
| 외로움 | Wide Shot, 24mm | Blue, 새벽 | Fog, Rain, Slow Camera |
| 공포 | Close Up | Red | Heartbeat, Camera Shake |
| 희망 | Wide | Golden, Sunrise | Slow Piano |
| 신비 | — | — | Fog |

예: Scene의 감정이 "외로움"이면 → Wide Shot → 24mm → Blue → Fog → Slow Camera → Prompt 자동 생성.

## 감정 수치 모델 (Emotion Engine 연동)

Scene마다 감정을 수치로 분석한다:

```
Scene: 인류 마지막 날
외로움 80 / 희망 10 / 공포 60 / 신비 70
```

가장 높은 감정이 연출을 지배하고, 보조 감정이 디테일을 만든다.

## Shot 다양화 규칙 (가장 흔한 실패 방지)

모든 장면이 같은 카메라면 영화가 아니다. **Shot은 계속 바뀐다.**

### ❌ 금지 패턴

```
뒤에서 사람 → 넓은 화면 → 뒤에서 → 넓은 화면 → 뒤에서 (반복)
```

### ⭕ 영화 패턴 (Scene 하나의 Shot 분해 예)

```
Shot 1  Wide (도시 전체)      5초
Shot 2  발 (걸어감)           2초
Shot 3  헬멧 클로즈업          3초
Shot 4  어깨 뒤 (Shoulder)    5초
Shot 5  드론샷               3초
Shot 6  눈 클로즈업           2초
Shot 7  기지 Wide            4초
```

넷플릭스의 흐름: 도시 → 발 → 손 → 헬멧 → 우주 → 기지 → 하늘 → 사람 → UI → 지도 (계속 변화)

## 영상 구성 비율 (25분 기준)

**25분을 전부 AI 영상으로 만들지 않는다. 오히려 조회수가 떨어진다.**

| 요소 | 비율 | 이유 |
|------|------|------|
| AI 영상 (Kling) | 20~25% | 감정과 임팩트가 필요한 장면만 |
| Midjourney 이미지 + 카메라 무빙 (패닝/줌/패럴랙스) | 50~60% | 가장 자연스럽고 비용 효율적 |
| 인포그래픽/지도/타임라인/UI | 10~15% | 정보 전달과 리듬 변화 |
| 텍스트·숫자·강조 연출 | 5~10% | 핵심 메시지와 몰입 유지 |

### 구간별 배분

| 구간 | 길이 | 구성 |
|------|------|------|
| Intro | 1분 | 100% 영상 (영화급) |
| 본론 | 15분 | 70% 이미지 + 30% 영상 |
| 클라이맥스 | 5분 | 70% 영상 + 30% 이미지 |
| 엔딩 | 4분 | 거의 이미지 + 음악 |

이유: 사람은 처음과 끝만 기억한다. 그리고 비용이 크게 절약된다 (300 영상 → 60 영상 + 240 이미지).

## 장면 전환(Scene Change)이 곧 프리미엄이다

사람들은 "영상"을 보는 게 아니라 **"장면 전환"** 을 본다.

```
5초 영상 → 2초 이미지 → 4초 영상 → 3초 이미지 → 6초 지도 → 5초 뉴스 화면 → 4초 영상 → 2초 위성사진
```

이것이 "30초 영상 하나"보다 훨씬 비싸 보인다.

## 비주얼 모티프 (Visual Motif)

매 영상 반복되는 요소로 채널 정체성을 만든다. Beyond Humanity의 모티프:

- AI가 만든 **지구 홀로그램**
- 미래 연도를 표시하는 **HUD 인터페이스**
- "Simulation Start..." 동일한 오프닝 애니메이션
- 같은 내레이터 톤 + 같은 배경 음악 테마
- 동일한 색감 (청록 + 주황 계열)

## 요약 공식

```
영상 = 영화 / 이미지 = 설명 / 인포그래픽 = 이해 / 텍스트 = 강조
```

모든 것을 움직이게 만드는 것이 목표가 아니다. **시청자의 감정이 움직이게 만드는 것이 목표다.**
