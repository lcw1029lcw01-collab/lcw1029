# ADOS Project Charter

Version: 0.2.0

> 프로젝트 헌장. 우리가 무엇을, 왜, 어떤 기준으로 만드는지 정의한다.

---

# 프로젝트명

ADOS (AI Documentary Operating System)

- 대외 정체성: **AI Documentary Studio** (운영체제는 내부 구조이고, 사용자는 스튜디오를 사용한다)
- ADOS는 회사, **Beyond Humanity는 첫 번째 브랜드(제품)** 다.

---

# 미션 (Mission)

AI가 영화 감독처럼 사고하고,
영화 수준의 다큐멘터리를 자동 제작하는 운영체제를 개발한다.

# 비전 (Vision)

단순한 AI 영상 생성기가 아니라

**AI 콘텐츠 제작 회사 전체를 자동화하는 플랫폼**을 구축한다.

---

# 성공 기준

사람은 아래 4가지만 결정한다.

1. 주제
2. 언어
3. 영상 길이
4. 채널

그 외 모든 과정은 ADOS가 수행한다.

---

# 우리가 만들지 않는 것

❌ 슬라이드쇼

❌ AI 느낌이 나는 영상

❌ 단순 이미지 나열

❌ 단순 Prompt 생성기

❌ 단순 대본 생성기

# 우리가 만드는 것

✔ 영화 같은 연출

✔ 영화 같은 카메라

✔ 감정 기반 스토리

✔ 세계관 유지

✔ 자동 편집

✔ 자동 업로드

---

# 핵심 가치

1. **Quality First** — 품질을 속도보다 우선한다.
2. **Story First** — 스토리가 먼저다.
3. **Emotion First** — 감정이 먼저다.
4. **Direction First** — 감독의 연출이 먼저다.
5. **Automation First** — 사람의 반복 작업을 없앤다.
6. **Modular** — 모든 기능은 독립적으로 교체 가능해야 한다.

---

# 개발 원칙

설계 후 개발

문서 후 코드

테스트 가능한 구조

AI가 읽기 쉬운 문서

JSON 중심 데이터

**YAGNI — 지금 필요하지 않은 것은 만들지 않는다.**

**문서는 코드보다 중요하다.** (좋은 문서는 모든 AI가 사용할 수 있지만, 좋지 않은 코드는 몇 달 후 버려질 수 있다)

| **Beyond Humanity** | 미래 문명·인류·과학·철학 | 1순위. 첫 에피소드 제작 대상 |
| **Humanity Explained** | 인간 심리·행동·진화 | |
