# ADR-0004: AI 직원(Agent) 협업 구조는 v1.0 이후로 보류

- 날짜: 2026-07-09
- 상태: 채택

## 결정

"AI 직원(Employee/Agent)들이 협업하는 회사 구조"는 초기 버전에서 구현하지 않는다.
단계적으로 도입한다:

| 버전 | 구조 |
|------|------|
| v0.1 (지금) | 단일 파이프라인 — 하나의 AI가 모든 의사결정 수행 |
| v0.5 | Research / Story / Director를 역할별 모듈로 분리 |
| v1.0 이후 | Research Agent, Director Agent, Editor Agent 등 여러 AI 협업 구조 |

이에 따라 `agents/` 폴더(직원 정의 5개)는 v1.0 최종 구조에서 제외했다.
(내용은 git 히스토리에 보존 — 커밋 `450818f` 참조. Agent 도입 시 복원해 재사용한다.)

## 이유

- 처음부터 Employee 구조로 설계하면 시스템이 너무 복잡해진다.
- 우리의 목표는 거대한 시스템이 아니라 **"끝까지 완성할 수 있는 시스템"** 이다.
- 처음부터 과도하게 복잡한 시스템을 만드는 위험을 피한다. (YAGNI — ADR-0003)
