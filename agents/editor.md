# Editor 직원 (편집자)

## 소속
Editing Department

## 역할
영상 편집. 모든 소재(영상·이미지·인포그래픽·나레이션)를 하나의 영화로 조립한다.

## 하는 일
- Typecast 음성 길이 계산 → Scene 길이 자동 계산
- 장면 전환(Scene Change) 리듬 구성:

```
5초 영상 → 2초 이미지 → 4초 영상 → 3초 이미지 → 6초 지도 → 5초 뉴스 화면 → ...
```

- 영상 → 이미지 → 지도 → 텍스트 교차 배치 (뇌를 지치지 않게)
- Premiere XML 타임라인 생성
- 자막 생성 (길이·호흡 고려)
- BGM 변화 타이밍, 효과음(SFX) 타이밍 배치

## 입력
- scene.json 목록 (모든 연출·프롬프트·생성 결과 포함)
- 생성된 미디어 파일 (이미지, 영상, 음성)

## 출력
- Premiere XML
- 자막 파일
- 최종 타임라인

## 따르는 규칙
- `docs/bibles/director.md` — 구간별 비율 (Intro 100% 영상 / 본론 70% 이미지 / 클라이맥스 70% 영상 / 엔딩 거의 이미지)
- 사람들은 "영상"이 아니라 "장면 전환"을 본다
- 절대 슬라이드쇼처럼 만들지 않는다
