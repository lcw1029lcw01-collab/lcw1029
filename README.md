# ADOS

**AI Documentary Operating System** — AI 다큐멘터리 운영체제

> **"AI가 감독처럼 생각하고, 영화처럼 제작한다."**

Netflix 수준의 AI 다큐멘터리를 자동 생성하는 운영체제.
주제 하나만 입력하면 리서치 → 대본 → 연출 → 이미지 → 영상 → 나레이션 → 편집 → SEO → 업로드까지 자동으로 수행한다.

## 목표

- 영화 수준의 연출
- AI 자동화 (Decision Engineering — Prompt가 아니라 결정을 자동화)
- 다국어 지원 (영어 + 한국어)
- 세계관 유지 (World Bible)
- 자동 편집
- 자동 업로드

## 시작하기 (AI와 사람 공통)

**`문서/00_AI지침.md`부터 번호 순서로 읽는다.**

```
문서/
├── 00_AI지침.md        ⭐ 가장 먼저
├── 01_프로젝트헌장.md   헌법
├── 02_비전.md            비전
├── 03_시스템구조.md      구조
├── 04_로드맵.md           로드맵
├── 05_용어집.md          용어집
├── 06_개발규칙.md 개발 규칙
└── 바이블/                 제작 규칙 (스토리·연출·카메라·감정·세계관·프롬프트)
```

현재 상태는 [현재상태.md](현재상태.md), 변경 이력은 [변경이력.md](변경이력.md) 참조.
